# 🍎 Stephen's Mac OS Portfolio

A highly interactive, OS-inspired portfolio website built with **Next.js 15**, **Tailwind CSS**, and **Lucide React**. This project simulates a desktop environment with windows, a functional dock, and integrated applications.

## 🚀 Live Demo
**[portfolio-link.vercel.app](https://mac-os-portofolio-dy26m8l7c-stephens-projects-d4a1057e.vercel.app?_vercel_share=D7a67D4KksnnLO9UhOmVqBxHrkCExp8C)** *(Ganti dengan link Vercel kamu setelah deploy)*

---

## ✨ Key Features

### 🖥️ Interactive Desktop Environment
- **Multi-Window System**: Open, close, and focus multiple apps (Terminal, Projects, About).
- **Functional Dock**: Smooth hover animations and active app indicators.
- **Terminal Emulator**: Navigate the portfolio using CLI commands or the GUI.

### 🤖 Featured Projects
- **Zyrex Customer Service RAG**: A specialized AI chatbot using **ColPali architecture** and **Guard Rails** to automate technical support for Zyrex device manuals.
- **Dering**: A high-security video communication platform for Indonesian correctional facilities.
- **AILO**: A local multimodal AI interface for privacy-focused assistance.
- **Real-Time Vehicle Tracking**: Fleet management system with live GPS data visualization.

### 🎮 Built-in Apps
- **Space Invaders**: A fully playable retro game built directly into the OS.
- **PDF Resume Viewer**: Seamlessly preview professional documents.

---

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **State Management**: Zustand (useDesktopStore)
- **Deployment**: Vercel

## 🛠️ Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/StephenJP14/mac-os-portofolio.git
   ```

2. **Install dependencies:**
   ```bash
   pnpm install
   ```

3. **Run the development server:**
   ```bash
   pnpm dev
   ```

4. **Open in browser:**
   Go to [http://localhost:3000](http://localhost:3000)
